Створення веб-сервера HTTP за допомогою Node.js

За допомогою модуля http в Node.js ми можемо створити власний веб-сервер HTTP. Це дозволяє нам обробляти HTTP-запити, відправляти HTTP-відповіді та створювати веб-додатки. Для створення веб-сервера ми використовуємо функцію createServer() модуля http, яка приймає функцію зворотного виклику (callback) для обробки запитів. У цій функції ми визначимо логіку обробки запитів та відправлення відповідей.
