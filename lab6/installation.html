Встановлення Node.js

Інструкція по встановленню

  1. Завантажте виконуваний файл для встановлення Node.js з офіційного веб-сайту (https://nodejs.org).
  2. Запустіть виконуваний файл для початку процесу встановлення. Виберіть версію Node.js, яку бажаєте встановити. Зазвичай рекомендується встановлювати останню стабільну версію.
  3. Після успішного встановлення ви можете перевірити, чи встановлений Node.js правильно, відкривши термінал або командний рядок і виконавши наступну команду:

    node -v

    Ця команда виведе версію Node.js, якщо встановлення пройшло успішно.
  4. Тепер ви можете створювати та виконувати Node.js-програми.
