Способи підключення стилів:

  1. Підключення зовнішнього CSS файлу за допомогою тега link. Це найбільш зручний і популярний спосіб. Він полягає в тому, що у тегу link задається атрибут href, в якому вказується шлях до CSS файлу.

    2. <head>
      <link rel="stylesheet" type="text/css" href="styles.css">
    </head>


  2. Вбудовування CSS коду безпосередньо у тег style. Цей спосіб використовують, коли потрібно задати стилі для конкретного елементу або групи елементів на сторінці. Він полягає в тому, що у тегу style вписується CSS код.

    <head>
      <style>
       h1 {
        color: red;
       }
       p {
        font-size: 14px;
       }
     < /style>
    </head>


  3. Використання атрибуту style безпосередньо у HTML елементів. Цей спосіб використовують, коли потрібно задати стилі лише для одного елементу.

    <p style="color: blue;">Цей текст буде синім кольором</p>
