Селектор атрибуту - вибирає елементи з вказаним значенням атрибуту.

CSS
a[href="https://www.google.com"] {
  color: red;
}

Приклад застосування

Селектор елементів-нащадків - вибирає всі елементи, які є нащадками безпосереднього батька.

CSS
div h5 {
  color: blue;
}

Приклад застосування

Селектор псевдокласу - вибирає стан елемента.

CSS
button:active{
  background: #c9c9c9;
}
